# دليل تشغيل IraqNow Bot على Termux (هواوي / أندرويد)

## 1. تنصيب Termux
- نزّل Termux من **F-Droid** (مو من متجر بلاي — النسخة هناك قديمة ومكسورة):
  https://f-droid.org/packages/com.termux/
- افتح Termux ونفّذ:
  ```
  termux-setup-storage
  pkg update -y && pkg upgrade -y
  pkg install -y python git
  ```

## 2. نقل ملفات البوت
- فك ملف الزب (telegram-bot.zip) وضع محتواه بمجلد داخل Termux، مثلاً:
  ```
  mkdir -p ~/iraqnow-bot
  # ثم نقل كل الملفات لهذا المجلد (عن طريق مدير الملفات أو unzip)
  cd ~/iraqnow-bot
  ```

## 3. تثبيت المكتبات
```
pip install -r requirements.txt
```
إذا واجهت خطأ بتنصيب حزمة معينة، جرب:
```
pkg install -y build-essential libffi-dev
```

## 4. ضبط المفاتيح
- انسخ `.env.example` باسم `.env`:
  ```
  cp .env.example .env
  ```
- افتح `.env` بأي محرر نصوص (`nano .env`) وضع:
  - `BOT_TOKEN`: التوكن من @BotFather
  - `ADMIN_ID`: آيدي حسابك بالتليكرام (احصل عليه من بوت مثل @userinfobot)
  - باقي المفاتيح اختيارية — البوت يعمل بدونها، فقط ميزات معينة (طقس، ذكاء اصطناعي) تتعطل إذا تركتها فاضية.

## 5. تشغيل البوت
```
termux-wake-lock
python main.py
```
`termux-wake-lock` يمنع أندرويد من تنويم Termux — لازم تشغّله قبل البوت.

## 6. مهم جداً بخصوص هواوي (EMUI) — منع قتل التطبيق بالخلفية
نظام هواوي يقتل التطبيقات بقوة لتوفير البطارية، وهذا يوقف البوت بعد إقفال الشاشة إذا ما ضبطت:
1. الإعدادات → البطارية → إطلاق التطبيقات (App launch) → دوّر Termux وفعّل "إدارة يدوية" ثم فعّل الثلاث خيارات (التشغيل التلقائي، التشغيل الثانوي، التشغيل بالخلفية)
2. الإعدادات → البطارية → تحسين استهلاك الطاقة → استثني Termux
3. تأكد الهاتف متصل بشبكة مستقرة (واي فاي أو بيانات) باستمرار

## 7. تشغيله بالخلفية بشكل دائم (اختياري لكن يُنصح به)
- ثبّت إضافة **Termux:Boot** من F-Droid (نفس مطور Termux) — تجعل البوت يشتغل تلقائياً عند إعادة تشغيل الهاتف.
- أو استخدم `tmux` حتى تقدر تسكّر شاشة Termux والبوت يواصل الشغل:
  ```
  pkg install -y tmux
  tmux new -s bot
  termux-wake-lock
  python main.py
  # اضغط Ctrl+B ثم D للخروج بدون إيقاف البوت
  # للرجوع: tmux attach -t bot
  ```

## استكشاف الأخطاء
- إذا البوت طلع فيه خطأ "المتغير الإلزامي 'BOT_TOKEN' غير مضبوط" → تأكد ملف `.env` موجود بنفس مجلد `main.py` وفيه التوكن صحيح.
- إذا توقف البوت عن الرد بعد وقت من إقفال الشاشة → رجع لخطوة 6 (إعدادات البطارية بهواوي).
